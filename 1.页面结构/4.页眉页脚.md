# header
`header` 标签用于定义文档的页眉。

```html
<body>
  <header>
    <nav>
      <ul>
        <li><a href="">菜单一</a></li>
        <li><a href="">菜单二</a></li>
      </ul>
    </nav>
  </header>
  <article>
    ...
  </article>
  ...
</body>
```

# footer
`footer` 标签定义文档或节的页脚，页脚通常包含文档的作者、版权信息、使用条款链接、联系信息等等。

```html
<body>
    ...
    <article>
      ...
    </article>
    <footer>
        <p>
            ...
        </p>
    </footer>
</body>
```
